Basic CSS

CSS (Cascading Style Sheets) is used to style and layout web pages. It controls colors, fonts, spacing, alignment, and responsiveness.

  • Selectors target HTML elements.
  • Properties define how elements are styled.
body { background-color: #f4f4f4; font-family: Arial, sans-serif; }

CSS rules consist of a selector and a declaration block enclosed in curly braces.

h1 { color: blue; }
Design

Design in CSS focuses on layout structure, spacing, visual hierarchy, and user experience.

  • Use whitespace effectively.
  • Maintain consistent color schemes.
.container { margin: 0 auto; max-width: 1200px; }

Good design ensures readability, clarity, and a pleasant user experience.

.card { border-radius: 8px; box-shadow: 0 4px 8px rgba(0,0,0,0.1); }
Absolute and Relative units

CSS uses different units to define sizes. Absolute units have fixed values, while relative units scale dynamically.

  • Absolute units: px, cm, in.
  • Relative units: %, em, rem, vw, vh.
p { font-size: 16px; }

Relative units are preferred in responsive design because they adapt to different screen sizes.

h2 { font-size: 2rem; }
Colors

CSS supports various color formats including named colors, HEX, RGB, and HSL.

  • HEX example: #ff5733
  • RGB example: rgb(255, 87, 51)
body { background-color: #ffffff; color: #333333; }

Using consistent color palettes improves visual harmony and branding.

button { background-color: hsl(200, 70%, 50%); }
Styling Forms

CSS allows customization of form elements such as inputs, buttons, and labels.

  • Use padding for better spacing.
  • Use border-radius for modern appearance.
input { padding: 8px; border: 1px solid #ccc; }

Proper styling improves usability and accessibility of forms.

button { background-color: blue; color: white; }
The Box Model

The CSS Box Model describes how elements are structured in terms of content, padding, border, and margin.

  • Content is the inner part of the element.
  • Padding adds space inside the border.
div { padding: 10px; border: 2px solid black; }

Using box-sizing: border-box simplifies layout calculations.

* { box-sizing: border-box; }
Flex Box

Flexbox is a layout system designed for arranging elements in a single direction (row or column).

.container { display: flex; justify-content: space-between; }

Flexbox makes alignment and spacing easier compared to traditional layouts.

  • Use justify-content for horizontal alignment.
  • Use align-items for vertical alignment.
.item { flex: 1; }
Typography

Typography controls font selection, size, spacing, and readability of text content.

body { font-family: 'Arial', sans-serif; }

Good typography improves user engagement and readability.

  • Use line-height for better readability.
  • Use font-weight for emphasis.
p { line-height: 1.6; }
Accessibility

Accessibility ensures websites are usable by people with disabilities.

img { alt: "Descriptive text"; }

Using semantic HTML and proper contrast improves accessibility.

  • Use proper heading structure.
  • Ensure sufficient color contrast.
button:focus { outline: 2px solid blue; }
Positioning

CSS positioning allows precise placement of elements on a page.

.box { position: relative; }

Position values include static, relative, absolute, fixed, and sticky.

  • Absolute positioning removes element from normal flow.
  • Fixed positioning attaches element to viewport.
.header { position: fixed; top: 0; }
Responsive Design

Responsive design ensures a website adapts to various screen sizes using flexible layouts and media queries.

  • Use fluid grids for layout flexibility.
  • Apply media queries for breakpoints.
@media (min-width: 768px) { .container { width: 750px; } }

Mobile-first design starts with small screens and enhances for larger devices.

img { max-width: 100%; height: auto; }
